# 领域知识

## 游戏业务
- **游戏名称**: 砺刃使者、勇者盟约、峡谷行动（FPS类型）
- **乐园**: 砺刃使者的UGC玩法模式集合
- **大盘**:
  - 手游大盘/平台大盘 = 所有游戏集合
  - 游戏大盘（如"砺刃大盘"）= 该游戏全部活跃用户
- **实体关系**: 1个玩家ID可对应多个角色ID
- **流水**: 指玩家行为明细日志或付费充值金额

## 指标定义
- **留存**: 当天活跃且次日仍活跃的用户（次留），其他留存类推
- **新进**: 注册用户
- **活跃**: 未明确玩法时，默认指游戏整体活跃，非单个玩法模式

## 数仓规范

### 分层
- `DWD`: 行为明细，一条事件一条记录
- `DWS`: 玩家粒度或聚合数据
- `DIM`: 维度配置表

### 命名规则（示例：`dws_jordass_mode_roundrecord_di`）
| 部分 | 含义 |
|------|------|
| `dws` | 数据分层 |
| `jordass` | 游戏代码 |
| `mode_roundrecord` | 业务含义 |
| `di` | d=按天分区, i=增量；df=全量（取周期末日） |

**DWD表命名**: `dwd_{gamecode}_{行为标识}_hi`

### 关键字段
- **cbitmap**: 100位二进制串，第1位=当天，1=发生行为，0=未发生；用于计算流失/回流/留存
